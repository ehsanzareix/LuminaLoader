# Design tokens

Lumina exposes a small set of CSS custom properties for easy theming and sizing.

- `--lumina-size` — base size (px)
- `--lumina-color` — accent color used by most loaders
- `--lumina-spin-speed` — spin/animation duration (s)
- `--lumina-ease` — easing curve used for animations

## Usage

Set tokens on `:root` or a container element to scope them.

```css
:root {
  --lumina-color: #4fa94d;
  --lumina-size: 48px;
  --lumina-spin-speed: 1s;
}
```
